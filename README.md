## Setup
1. Pastikan ```GNU Prolog``` telah terinstall
2. Clone repository ini pada local computer anda
3. Masuk ke dalam folder hasil clone repository
4. Jalankan ```main.pl```
5. Masukkan ```start.``` untuk memulai permainan.
